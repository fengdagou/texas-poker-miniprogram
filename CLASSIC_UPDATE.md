## 🎯 技术细节

### CSS 特性

```css
/* 木质纹理背景 */
background: 
  repeating-linear-gradient(
    90deg,
    rgba(139, 69, 19, 0.08) 0px,
    rgba(139, 69, 19, 0.08) 2px,
    transparent 2px,
    transparent 4px
  ),
  linear-gradient(135deg, #2c1810 0%, #1a0f0a 50%, #2c1810 100%);

/* 金色边框 */
border: 3rpx solid rgba(212, 175, 55, 0.4);

/* 古典阴影 */
box-shadow: 
  0 8rpx 30rpx rgba(0, 0, 0, 0.5),
  inset 0 2rpx 15rpx rgba(212, 175, 55, 0.1);

/* 文字阴影 */
text-shadow: 0 2rpx 15rpx rgba(212, 175, 55, 0.4);
```
