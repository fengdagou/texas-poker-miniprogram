## 🎬 新增功能

### 1. 荷官 Victoria 登场

**角色设定：**
- 名字：Victoria（维多利亚）
- 形象：金发碧眼、优雅专业的荷官
- 风格：古典奢华、迷人气质

**出现时机：**
- 每局游戏开始时
- 发牌阶段自动登场

### 2. 发牌动画

**动画效果：**
- 3 张卡牌飞舞效果
- 卡牌从荷官方向飞向玩家
- 流畅的抛物线轨迹
- 2.5 秒完整动画

**发牌消息：**
随机显示以下消息之一：
- "发牌中..."
- "祝你好运 🍀"
- "精彩的对局即将开始"
- "Victoria 为你发牌"
- "准备好迎接好运了吗？"

## 🔧 技术实现

### 数据结构
```javascript
data: {
  isDealing: false,      // 是否正在发牌
  dealingMessage: ''     // 发牌消息
}
```

### 动画时序
```javascript
showDealingAnimation() {
  // 显示动画
  this.setData({ isDealing: true })
  
  // 2.5 秒后隐藏
  setTimeout(() => {
    this.setData({ isDealing: false })
  }, 2500)
}
```

### CSS 动画
- `dealerFadeIn` - 荷官淡入
- `rotate` - 光环旋转
- `messagePulse` - 消息脉冲
- `flyCard1/2/3` - 卡牌飞舞
