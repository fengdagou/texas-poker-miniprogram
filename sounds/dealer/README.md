## 语音风格要求

**荷官 Victoria 的声音特点：**
- 女声，优雅专业
- 语速适中，清晰悦耳
- 略带磁性，有亲和力
- 古典赌场荷官风格

## 生成方式

### 方式 1：使用 TTS 工具（推荐）

使用 OpenClaw 的 TTS 功能生成：

```
/tts 发牌中
/tts 祝你好运
/tts 精彩的对局即将开始
/tts Victoria 为你发牌
/tts 准备好迎接好运了吗
```

生成的文件复制到本目录，并重命名为 `dealer-1.mp3` 到 `dealer-5.mp3`

### 方式 2：真人录制

找一位声音优雅的女性录制以上 5 句话，导出为 MP3 格式。

### 方式 3：在线 TTS 服务

使用以下服务生成：
- 阿里云智能语音
- 百度语音合成
- 腾讯云语音合成
- ElevenLabs（英文）

推荐参数：
- 音色：温柔女声
- 语速：1.0（正常）
- 音调：中等
- 音量：80%

## 技术要求

- **格式**: MP3
- **采样率**: 44.1kHz 或 48kHz
- **比特率**: 128kbps 或更高
- **时长**: 每条约 1-3 秒
- **文件大小**: 每条约 10-50KB
