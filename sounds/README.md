# 🎵 音效资源说明

## 需要的音效文件

请将以下音效文件放入 `sounds/` 目录：

### 背景音乐
- `bgm.mp3` - 轻松愉快的赌场背景音乐（循环播放）
  - 推荐时长：2-3 分钟
  - 风格：轻爵士/赌场氛围

### 音效
- `deal.mp3` - 发牌音效（卡牌飞出的声音）
  - 时长：0.5 秒
  
- `bet.mp3` - 下注音效（筹码放置声）
  - 时长：0.3 秒
  
- `win.mp3` - 胜利音效（庆祝音乐）
  - 时长：2-3 秒
  
- `lose.mp3` - 失败音效（低沉音效）
  - 时长：1-2 秒
  
- `chip.mp3` - 筹码碰撞声
  - 时长：0.2 秒
  
- `fold.mp3` - 弃牌音效
  - 时长：0.3 秒
